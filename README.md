# 📧 Email/SMS Spam Classifier

A machine learning web app that classifies emails and SMS messages as Spam or Not Spam using NLP and Multinomial Naive Bayes.

## 🎯 Results
- **Accuracy:** 98.4%
- **Precision:** 100%
- **Dataset:** 5,572 labelled messages

## 🛠️ Tech Stack
- Python, Scikit-learn, NLTK
- TF-IDF Vectorization
- Streamlit (Web App)

## 🚀 Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
```

## 📊 Models Compared
12 models tested — voting classifier(SVM,MNB,RF) selected after model improvement for 100% Precisionand better acuraccy
